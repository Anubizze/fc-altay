Vercel (репозиторий с приложением в папке frontend)

1. Vercel → Project → Settings → General → Root Directory: frontend
   (обязательно, иначе не найдёт Next.js и lockfile.)

2. Build / Install оставь по умолчанию или как в frontend/vercel.json
   (pnpm install --frozen-lockfile, pnpm run build).

3. После смены Root Directory сделай Redeploy.
